# Дизайн-Обзор kAIgo

Этот документ фиксирует текущий продакшен-дизайн kAIgo и описывает структуру страниц, секций, визуальной системы и пользовательских сценариев.

## Обзор

kAIgo - темный glassmorphism-сайт портфолио AI-разработчика. Интерфейс сочетает фиксированную навигацию, анимированный particle-фон, сине-фиолетовые градиенты, стеклянные панели и сфокусированные CTA. Сайт показывает AI-услуги, измеримые кейсы, блоки доверия и контактные сценарии для обсуждения проектов.

## Страницы И Маршруты

| Маршрут | Назначение |
| --- | --- |
| `/` | Главная лендинговая страница с hero, услугами, проектами, преимуществами, отзывами, контактами и AI-виджетом. |
| `/about` | Страница профиля: специализация Никиты в AI, подход, метрики, компетенции и CTA для сотрудничества. |
| `/projects` | Сетка портфолио с фильтрами, статусами, тегами и переходами в детальные кейсы. |
| `/project/:id` | Детальная страница кейса: контекст, задача, решение, технологии, результаты и навигация. |
| `/auth` | Вход, регистрация и сброс пароля через Supabase Auth. |
| `/login` | Отдельный вход администратора с восстановлением пароля и редиректом для admin-пользователей. |
| `/admin` | Защищенная админ-зона для заявок, истории диалогов, AI-настроек и управления доступом. |

## Визуальная Система

- Темный фон с анимированным particle-полотном.
- Стеклянные панели с мягкими границами и blur-эффектом.
- Сине-фиолетовые градиенты для заголовков, иконок, кнопок и метрик.
- Фиксированная верхняя навигация: Главная, О нас, Проекты, Контакты.
- Основной CTA построен на ярко-синих округлых кнопках.
- Ритм секций основан на крупных центрированных заголовках, компактном описании и карточных сетках.

## Технический Стек

- Frontend: Vite, React 18, TypeScript, React Router, Tailwind CSS.
- UI: shadcn/ui, Radix UI primitives, lucide-react, Sonner/Radix toasts, кастомные glass- и animation-стили.
- Данные и auth: Supabase Auth, Supabase Database, типизированный клиент `@supabase/supabase-js`, role-aware admin flow.
- Админка: Recharts для аналитики, date-fns для форматирования дат.
- AI: Supabase Edge Functions `ai-chat` и `voice-to-text`, VseGPT/OpenAI-compatible настройки моделей, конфигурация вроде `openai/gpt-4o-mini`, транскрибация `whisper-1`.
- Контакты: Supabase Edge Function `send-contact-email`, таблица `contact_requests`, отправка email через Resend.
- Сессии: localStorage хранит короткую историю AI-чата на 30 минут.
